## Quick Reference

### Installation
```bash
cd /workspace/TaskVaultLite
npm install
npm run dev
```

### Project Location
```
/workspace/TaskVaultLite
```

### Main Features
- Create tasks with title, description, links, images
- View tasks in responsive grid
- Click tasks to see full details
- All data in localStorage
- Works offline

### Tech Stack
- React 19.2.0
- Vite 7.3.1
- CSS3
- localStorage
- FileReader API

### Storage
- Key: `taskvault_tasks`
- Format: JSON
- No backend needed
